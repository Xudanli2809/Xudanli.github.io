实验三 使用Github管理Android项目

一、实验目的

1. 掌握Android Studio中Git使用

2. 掌握Github管理项目

二、实验任务

1. 在Android Studio中新建项目

2. 添加VCS中的Git支持

3. 发布项目到Github

4. 在Android Studio中进行修改代码、提交、新建分支、合并等操作

5. 推送项目到Github

6. 如有多人合作，处理pull request

三、实验步骤与结果

1.在Android Studio里新建项目，我的项目是手账APP，所以取名为HandAccount。

2.添加VCS中的Git支持。点击菜单栏VCS—>Import into Version Control—>Share Project on Github，然后填写自己的github账号和密码，如下图所示

3.添加完Git后，VCS栏中多了Git选项如图所示

4.发布项目到Github。点击Git—>Push，即可发布到Github上。发布完成后如下图所示

5.在Android Studio里修改代码。这里我在activity_hand_account.xml中修改了”忘记密码“按钮的显示文本。同时，可以看到相应的被改过的文件变成了蓝色。提交时，点击Commit Changes，填写好Commit Message，点击Commit and Push，即可提交更改。

6.最后github及AS上的结果如下

四、实验小结

在Android Studio里添加Git功能后，使上传项目更加方便，即将原先复杂的过程集成到一起，方便又快捷。可以实现在不同电脑上对项目的编辑及保存，相当于一个云端，防止项目的丢失。
